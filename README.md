# customer_behavior_analysis
Data analytics and visualization project showcasing customer sales behavior pattern using Python, Postgres and PowerBI

## Analysis Steps
1. **Data Loading**
   - Imported dataset into VS Code
   - Standardized column names and data types

2. **Exploratory Data Analysis (EDA)**
   - Distribution analysis
   - Missing value assessment
   - Category frequency and imbalance checks
   - Identification of key metrics and drivers

3. **Data Cleaning**
   - Handled null values based on domain logic
   - Removed or corrected inconsistent records
   - Engineered derived variables where required

4. **SQL Analysis**
   - Loaded cleaned data into a relational database Postgres
   - Executed analytical queries using joins, aggregations, and filters
   - Validated Python results using SQL

5. **Power BI Dashboard**
   - Built an interactive dashboard for stakeholders
   - Implemented KPIs, trend analysis, and drill-downs
   - Focused on clarity, usability, and decision support

6. **Reporting & Presentation**
   - Summarized findings, assumptions, and limitations
   - Delivered insights via a structured report
   - Created an executive-ready presentation using Gamma
